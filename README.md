**SelectoresApp** es una aplicación Angular 14 que implementa un sistema de selectores en cascada para la selección interactiva de países y regiones. El proyecto demuestra el uso de formularios reactivos de Angular combinados con RxJS para crear tres dropdowns dependientes: primero se selecciona un continente (África, América, Asia, Europa u Oceanía), luego se carga dinámicamente la lista de países de esa región, y finalmente se muestran los países fronterizos del país seleccionado. La aplicación consume la API REST Countries v2 para obtener información geográfica en tiempo real, utiliza operadores RxJS avanzados como `switchMap` y `combineLatest` para manejar las consultas asíncronas encadenadas, e implementa patrones modernos de Angular como módulos lazy-loaded, servicios para abstracción de datos, y manejo de estados de carga. Es un excelente ejemplo educativo de cómo construir selectores dependientes con Angular utilizando buenas prácticas de arquitectura y programación reactiva.
